## 🛠️ Cara Compile dan Run

### Menggunakan g++ (Command Prompt / Terminal)

Compile:

- g++ nama_file.cpp -o output

Run:

- ./output

Contoh:

- g++ soal-1.cpp -o soal1
- ./soal1
